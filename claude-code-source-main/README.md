## 🔧 如何使用

### 1. 下载源代码

**点击右侧绿色的 `<> Code` 按钮 → 在弹窗最下方点击 `zip download` 即可下载源码**

或者直接下载本仓库中的 `claude-code-source-main/` 文件夹。

### 2. 安装依赖

```sh
cd claude-code-source-main
npm install
```

### 3. 全局安装

```sh
npm install -g .
```

### 4. 开始使用

```bash
oh
```
