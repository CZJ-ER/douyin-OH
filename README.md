## 🔧 如何使用

### 1. 下载源代码

**点击右侧绿色的 `<> Code` 按钮 → 在弹窗最下方点击 `zip download` 即可下载源码**

打开随便一个ai助手，然后让他根据“RENAME_GUIDE.md”把名字改回去

然后就是完整的客户端了
